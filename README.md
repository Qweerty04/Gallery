# Gallery
This project is gallery of images
Task description:

Create a simple image gallery using the Vue framework.js. This task will help us evaluate your ability to work with Vue.js and organize code using components.

Tasks:

- Display a list of images with their name and author.
- The ability to click on the image to see it in an enlarged size.
- Ability to return to the list of images from the magnified view mode.

Requirements:

- Use Vue.js for development.
-Organize your code using Vue components.
- Apply CSS to style the gallery.
- Implement the functionality of switching between the list of images and the magnified view mode.
